# PSM Party Mode

## What

PSM Party Mode là một multi-agent discussion workflow nơi 4 production engineering experts (Khang, Minh, Ha, Linh) cùng thảo luận và debate về topics bạn đưa ra.

## How

1. **Agent Loading**: Tải đầy đủ PSM agent roster với Vietnamese-English personas
2. **Discussion**: Orchestrate multi-agent conversation với intelligent agent selection, cross-talk, và authentic character portrayal
3. **Graceful Exit**: Farewell sequence với session highlights

Mỗi agent có personality riêng:
- 🏗️ **Khang** — architecture-first, trade-off mindset
- 🚀 **Minh** — metric-driven, SLO-first
- 🔒 **Ha** — zero-trust, security-first
- 🤖 **Linh** — data-driven, experimental

## Why/Outcome

- Get diverse expert perspectives on production engineering topics
- Surface trade-offs and considerations from multiple viewpoints
- Generate creative solutions through agent debate
- Learn from agent cross-talk and disagreement

## When

Use when:
- User says "party mode", "team discussion", "multi-agent", "thảo luận với team"
- User wants to brainstorm with all PSM experts
- Complex topic needing multiple domain perspectives
